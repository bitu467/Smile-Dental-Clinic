# Smile Dental Clinic

A premium, modern web application for **Smile Dental Clinic**. This platform is designed to provide a seamless experience for patients and an efficient management system for clinic administrators.

## ✨ Features

- **Responsive Landing Page**: A stunning, mobile-friendly interface showcasing clinic services and expertise.
- **Admin Dashboard**: Secure management system for handling clinic data and patient inquiries.
- **Before & After Gallery**: Interactive slider to showcase dental treatment results.
- **Firebase Integration**: Powered by Firebase for hosting, real-time database, and secure storage.
- **SEO Optimized**: Built with best practices for search engine visibility.

## 🚀 Technologies Used

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend/Infrastructure**: 
  - Firebase Hosting
  - Firestore (NoSQL Database)
  - Firebase Storage
- **Design**: Premium typography, custom CSS animations, and high-quality imagery.

## 📂 Project Structure

- `index.html`: Main landing page.
- `admin.html`: Secure administrative interface.
- `styles.css`: Core design system and layout.
- `script.js`: Client-side logic and interactive elements.
- `admin.js`: Backend logic for the administrative dashboard.
- `firebase.json` & `firestore.rules`: Firebase configuration and security rules.

## 🛠 Setup & Deployment

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bitu467/Smile-Dental-Clinic.git
   ```
2. **Install Firebase CLI** (if not already installed):
   ```bash
   npm install -g firebase-tools
   ```
3. **Deploy to Firebase**:
   ```bash
   firebase deploy
   ```
